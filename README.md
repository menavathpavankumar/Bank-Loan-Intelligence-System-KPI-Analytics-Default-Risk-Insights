 Bank-Loan-Intelligence-System-KPI-Analytics-Default-Risk-Insights

->Overview
This project analyzes bank loan data to evaluate lending performance, borrower risk, and repayment behavior. It focuses on transforming raw financial data into meaningful insights that support data-driven lending decisions.

->Objective
The objective of this project is to:
- Measure overall loan portfolio performance
- Identify good and bad loans based on repayment behavior
- Analyze borrower financial health using key metrics
- Detect trends across time, region, and borrower characteristics

->Key Metrics
- Total Loan Applications  
- Total Funded Amount  
- Total Amount Received  
- Average Interest Rate  
- Average Debt-to-Income Ratio (DTI)  

->Loan Classification
Loans are categorized into:
- Good Loans: Stable repayment behavior  
- Bad Loans: Higher risk of default  

This classification helps in understanding risk distribution and financial exposure.

->Analysis Performed
- Monthly trend analysis of loan applications  
- Regional analysis by state  
- Loan term distribution  
- Employment length impact on lending  
- Loan purpose breakdown  
- Home ownership analysis  

->Key Insights
- High debt-to-income ratios are associated with increased default risk  
- Certain regions show higher loan volumes with weaker repayment patterns  
- Bad loans contribute disproportionately to financial risk  
- Borrowers with longer employment history tend to have better repayment behavior  

->Tools and Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Jupyter Notebook
